# Stake Affiliate Starter Kit

这是一个最小可用的 Stake 联盟推广基础包，目标是先帮你建立真实推广渠道，而不是空口去找客服申请专属。

当前落地页使用的活动链接：

```text
https://stake.com/?c=Supervip8
```

当前公开页面：

```text
https://lxq32467.github.io/stake-supervip8-landing/
```

## 里面有什么

- `index.html`：可直接打开的中文落地页。
- `assets/hero-gaming-desk.png`：页面头图，无 Stake 标志、无承诺收益文案。
- `30-day-content-calendar.md`：30 天内容发布计划。
- `outreach-templates.md`：联系小渠道、群主、创作者的私信模板。
- `campaign-plan.md`：活动拆分、渠道命名和执行顺序。
- `stake-campaign-tracker.csv`：推广数据追踪表。

## 后续如果要换活动链接

当前 `index.html` 已经替换成你的活动链接。以后如果要换成新的 campaign，打开 `index.html`，搜索：

```text
https://stake.com/?c=Supervip8
```

替换成新的 Stake 活动链接即可。

建议不要所有渠道都用同一个链接。你可以在 Stake 后台创建多个 campaign，例如：

- `telegram-main`
- `x-content`
- `youtube-shorts`
- `creator-001`
- `sports-event`
- `reactivation`

然后根据投放渠道替换页面链接，复制出不同版本。

## 上传方式

最简单的方式：

1. 用 Netlify、Vercel、Cloudflare Pages、GitHub Pages、Carrd 或自己的服务器上传整个文件夹。
2. 确保 `index.html` 和 `assets` 文件夹在同一级目录。
3. 打开页面测试按钮是否跳转到你的 Stake 活动链接。

## 合规原则

- 只面向成年人和 Stake 允许服务的地区。
- 不诱导使用 VPN 或虚假身份。
- 不承诺稳赚、回血、翻倍、包赢或投资收益。
- 不向未成年人、高风险人群或限制地区推广。
- 所有奖励和活动以 Stake 实际页面显示为准。

## 建议节奏

先用这个页面跑 14-30 天，积累截图和数据：

- 页面访问量
- 渠道关注人数
- 内容发布截图
- Stake campaign 的推荐用户、FTD、活跃数据
- 合作小渠道的数据反馈

之后再找 Stake affiliate team，会比现在直接申请专属更有说服力。
